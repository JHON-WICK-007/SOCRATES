# Custom Agent Rules

This file defines project-specific rules, style guidelines, and behavioral constraints for the agent.

## User Interface & Design Rules

- **Do Not Use Raw Emojis as UI Icons**: Never use raw Unicode emojis (e.g., 🔒, 🚀, ⭐, 📁, etc.) as icons in the website's user interface. Instead, use clean, professionally designed vector/SVG icons, icons matching the design system, or modern icon libraries.
